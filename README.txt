# Video Game Sales Dashboard

[Live  Demo] (https://video-game-sales-dashboard.onrender.com/)

## Overview
A data analytics dashboard built with Python and Dash that explores video game sales trends from 1980 to 2016.

## Questions Explored
- What genres are becoming more popular over the years?
- Which publishers have seen the most growth in global sales over time?
- Which regions have the largest growing video game market?

## Tools Used
- Python
- Pandas
- Plotly
- Dash

## How to Run
1. Clone the repo
2. Install dependencies: `pip install pandas plotly dash`
3. Download the dataset from [Kaggle](https://www.kaggle.com/datasets/gregorut/videogamesales) and place it in the `data/` folder
4. Run `app.py`

## Dataset
VGSales dataset by Gregory Smith, available on Kaggle.
